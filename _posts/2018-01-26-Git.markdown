### Git指令
~~~
git status
~~~
* 查看【本地】当前状态

~~~
git log
~~~
* 查看【本地】日志

~~~
git log origin/master
~~~
* 查看【远程】`master`分支的日志

~~~
git clone url
~~~
* 把【远程】仓库取到【本地】
