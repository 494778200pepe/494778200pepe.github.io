> 状态码的职责是当客户端向服务器端发送请求时，描述返回的请求结果。

![http_code]({{ site.baseurl }}/assets/images/Http/http_code.png)

### **2XX 成功**
2XX的响应结果表明请求被正常处理了。
#### 204 No Content
该状态码代表服务器接收的请求已成功处理，但在返回的响应报文中不含实体的主体部分。
##### 206 Partial Content
该状态码表示客户端进行了范围请求，而服务器成功执行了这部分`GET`请求。响应报文中包含由`Content-Range`指定范围的实体内容。
