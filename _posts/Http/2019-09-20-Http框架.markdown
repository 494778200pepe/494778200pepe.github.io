### **Http 功能**

* 支持http协议get、post、put、delete
* 异步请求
* 请求错误统一处理
* 支持多文件上传
* 预处理服务器返回的数据 converter（Json、Xml）
* 上传/下载 进度更新
* 取消请求
* 可扩展需求：
* timeout重试机制
* 多任务队列
* 关联activity
* 缓存刷新机制
* 请求方式：强制网络、先缓存后网络、先网络后缓存、仅缓存
* 缓存设置：内存缓存、数据库缓存、文件缓存(DiskLruCache)、SP缓存
