### 选择排序
```
    public static void selectSort(int[] data) {
        if (null == data || data.length == 0) {
            return;
        }
        int k;
        int j;
        for (int i = 0; i < data.length; i++) {
            k = i;
            for (j = i + 1; j < data.length; j++) {
                if (data[k] > data[j]) {
                    k = j;
                }
            }
            if (k != i) {
                int temp = data[i];
                data[i] = data[k];
                data[k] = temp;
            }
        }
    }
```
